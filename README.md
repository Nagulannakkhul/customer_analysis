Data analytics project of  Customer_Analysis using SQL.
# Customer Value & Discount Effectiveness Analysis

---Objective
Analyze customer lifetime value and evaluate whether discounts drive long-term profitability.

---Tools
- PostgreSQL (CTEs, Window Functions)

---Key Insights
- Discounted customers generate lower long-term profit
- High-value customers do not require discounts

---Business Impact
Helps pricing and marketing teams optimize discount strategy.
